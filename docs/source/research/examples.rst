==================
Example Notebooks
==================

The package includes tutorial notebooks under ``haymaker/research/examples``.
They use the local Arctic NQ data source, but the same workflow applies to any
dataframe with compatible OHLC columns.

Downloadable notebooks:

* :download:`Backtester basics <../../../haymaker/research/examples/01_backtester_basics.ipynb>`
* :download:`Stop-loss workflow <../../../haymaker/research/examples/02_stop_loss.ipynb>`
* :download:`Data bootstrapping with block bootstrap <../../../haymaker/research/examples/03_bootstrap_block.ipynb>`
* :download:`Data bootstrapping with regime states <../../../haymaker/research/examples/04_bootstrap_regime.ipynb>`

These notebooks are a good starting point for the end-to-end workflow:
loading data, generating simple strategy columns, grouping and upsampling,
running stop logic, calculating performance, and generating synthetic data.
